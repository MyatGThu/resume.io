# Off-the-clock screenshots

Drop screenshot images here and they appear automatically in the hover
popups on the **About → Off the clock** cards. Until a file exists, that
tile shows a clean branded placeholder, so the page never looks broken.

Each card reads three images. Use these exact filenames (`.jpg`, landscape,
~4:3, ideally ~800×600 or larger):

| Card             | Files                                            |
| ---------------- | ------------------------------------------------ |
| Dota 2           | `dota2-1.jpg`, `dota2-2.jpg`, `dota2-3.jpg`       |
| FIFA             | `fifa-1.jpg`, `fifa-2.jpg`, `fifa-3.jpg`          |
| Superhero films  | `heroes-1.jpg`, `heroes-2.jpg`, `heroes-3.jpg`    |
| The binge list   | `shows-1.jpg`, `shows-2.jpg`, `shows-3.jpg`       |

Commit the files to this folder (same way the company logos were added) and
push — the deploy picks them up on the next build.
